# connect_four
Challenge to implement a connect four game

# Prerequisites
- C++-20
- CMake 3.28.3
- GoogleTest

Can be installed with the following steps:
- "apt install -y build-essential cmake"
- "git clone https://github.com/google/googletest.git"
- "cd googletest"
- "mkdir build"
- "cd build"
- "cmake .."
- "make"
- "sudo make install"

# Run the app
Execute the following steps to build and run the connect four game:
- go in the src folder and execute the following commands
- "mkdir build"
- "cd build"
- "cmake .."
- "make"
- "./connect_four"

# Run the tests
Execute the following steps to build and run the unit tests:
- go in the tests folder and execute the following commands
- "mkdir build"
- "cd build"
- "cmake .."
- "make"
- "./connect_four_test"
